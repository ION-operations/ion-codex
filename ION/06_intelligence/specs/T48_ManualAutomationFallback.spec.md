# T48 — Manual Automation Fallback

Any automation-carried step must be manually carryable by a capable executor using the same lawful state, packet family, and landing rules.

Fallback must:
- preserve bounded-step law,
- preserve proposal-return law,
- preserve governed landing,
- and record that fallback occurred where relevant.
