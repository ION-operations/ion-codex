# Pass 25 — Project control kernel escalation

## What changed

Pass 25 adds the smallest missing layer on top of the default project control kernel:
- an escalation matrix
- an escalation judgment
- a project control kernel lifecycle

## Why it matters

The project now has an explicit answer to the next practical question after entering the default project control kernel:
- when may work stay inside the kernel
- when must it widen
- when must it stop and reassess

This reduces another historical failure mode: using the control kernel to start lawfully, but then drifting because there is no explicit rule for whether the kernel remains sufficient.
