# Atlas Packet Gate Checklist

A future work packet should not proceed unless all boxes can be checked.

- [ ] The work question is classified.
- [ ] The atlas base was consulted first.
- [ ] The primary authority surface is named explicitly.
- [ ] Relevant conflicts were checked.
- [ ] A work posture was chosen.
- [ ] A non-reinvention claim was made.
- [ ] The landing boundary is explicit.
- [ ] The starting center was recorded.
