# Clarification-only kernel revision example

## Trigger
A repeated misunderstanding exists about whether the conflict register must be checked before or after the defaults matrix.

## Why thaw is allowed
The problem is a local ambiguity inside the frozen kernel, not a need for a new center or wider organism change.

## Change class
`clarification_only`

## Widening
Not required.

## Expected result
Clarify ordering language inside the frozen kernel surfaces, then re-freeze.

## Why this example matters
It shows that not every kernel change is large; some are purely control-clarifying and should stay narrow.
