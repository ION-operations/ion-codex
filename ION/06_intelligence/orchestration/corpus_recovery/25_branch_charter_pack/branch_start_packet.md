# Branch start packet template

Use this exact packet when opening a new branched chat.

## Shared state

- canonical runnable root: `ION/`
- last completed objective: activation/lifecycle active-law emission
- current selected lane: Lane C runtime/session/API reintegration
- current horizon: review-entry and first promotion-candidate readiness
- out of bounds: active-law emission outside conductor; widening into Lane D; silent root-surface mutation

## Branch packet fields

### 1. Branch role
Name the branch as one of:
- conductor
- lane_c_runtime
- lane_a_meta_template
- adversarial_audit

### 2. Objective
State one bounded objective only.

### 3. Allowed files
List exact files or folders.

### 4. Forbidden files
List exact files or folders.

### 5. Expected output class
Choose one valid output class from the branch output contract.

### 6. Stop condition
State what counts as success or why the branch must hand back control.

## Reminder

A branch chat may propose.
Only the conductor may promote or emit.
