## Purpose

This file names the **smallest default operating kernel** for future project work.

Use this when you do not need the full atlas first and need the minimum lawful control path.

## Expansion rule

Only expand beyond this kernel when one of the following is true:
- the defaults matrix points to a preserved center you have not yet read
- the conflict register shows a conflict that must be compared directly
- the question class or posture is still unclear
- the work cannot proceed without the horizon/completion layer
- the task specifically requires one of the demonstration/example surfaces

## Why this exists

The project repeatedly failed when the control layer was:
- too diffuse to start from quickly
- too easy to bypass with plausible local framing
- too dependent on memory of the wider atlas

This default kernel is intended to be the smallest durable control path that still preserves:
- question discipline
- center discipline
- posture discipline
- anti-reinvention discipline
- conflict awareness
- packet-gate discipline
