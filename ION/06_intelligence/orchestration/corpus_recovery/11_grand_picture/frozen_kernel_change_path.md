## Principle

Treat the frozen kernel as the default baseline.
Do not thaw it casually.
If revision is truly needed, use the declared thaw path and require a re-freeze outcome.


## Why this path is fuller now

The frozen kernel now has not only a thaw protocol, but also an approval matrix, a revision lifecycle, and worked revision examples. This reduces the chance that kernel change proposals drift into vague or oversized control rewrites.
