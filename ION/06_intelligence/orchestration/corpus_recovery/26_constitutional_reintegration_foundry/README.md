# Constitutional Reintegration / Canon Foundry Suite

This directory is a planning and adjudication suite for the next layer of the
recovery program.

It does **not** replace the current active board, active startup authority, or
selected Lane C horizon. It exists because the branch now needs a disciplined
explanatory and programmatic layer for the harder question beneath the current
lane:

how should the wider ION organism be lawfully reintegrated without creating a
new false canon or flattening witness matter into implementation?
