# Soul coherence audit

## First-pass judgment
ION's core values were not erased, but their precedence order was broken.

The result was:
- real local recoveries
- real law/template restoration in some lines
- real runtime or manager centers in other lines
- but no single honest living center carrying the whole organism

## Soul-loss thesis
The project repeatedly substituted:
- local coherence for global recovery
- carrier convenience for truename truth
- partial branch normalization for full fractured-core reintegration

## First-pass recovery requirement
Any real repair must start with:
1. smallest values constitution
2. fractured-core map
3. explicit activation authority recovery
4. template -> context-bank loop recovery
5. runtime/API/session reintegration
6. manager/orchestrator/swarm reintegration
