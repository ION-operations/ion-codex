# Quarantined active-law review

This layer exists for **candidate active-law drafts** that are important enough to write, but not yet lawful to install into `ION/02_architecture/`.

Use this layer when:
- a reintegration lane has both a delta packet and a proposal-space design landing,
- a concrete draft is needed to expose hidden ambiguities,
- but active-law ratification would still be premature.

This layer is therefore:
- more concrete than `14_surface_design/`,
- less authoritative than `02_architecture/`,
- and explicitly subject to frozen-kernel review, thaw, and revision control.

## Boundary

Nothing in this layer is active law merely because it is written clearly.
A draft becomes active only through explicit review, approval, and installation into the live architectural spine.
