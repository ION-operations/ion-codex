# v2.6 Action Schemas

Use the subdirectories as authoritative:

```text
action_gateway/openapi.yaml
mcp_json_rpc/openapi.yaml
```

The root of this folder may contain copied reference files for convenience, but
the Action schemas to paste into GPT Actions are the two subdirectory
`openapi.yaml` files.

Both Actions are explicit-use only. They are not the default ION mount path.
Uploaded package/sandbox files are the default for ordinary Custom GPT work,
`/guest-mode`, `/what is ION?`, first-time context, and requests to use files or
instructions.
