# BUILDER READY INDEX

This directory contains the first builder-ready custom-GPT assembly pack for ION.

## Contents

- `01_BUILDER_READY_INSTRUCTIONS.md` — pasteable custom GPT instruction text
- `02_CONVERSATION_STARTERS.md` — suggested conversation starters for the GPT builder
- `03_BUILDER_WORKFLOW.md` — step-by-step builder procedure
- `04_KNOWLEDGE_PACK_MANIFEST.md` — curated Knowledge-pack selection and rationale
- `05_OPENAI_SURFACE_NOTES.md` — current platform-aware notes for GPT Knowledge, Data Analysis, and Actions posture
- `knowledge_pack/` — files to upload into GPT Knowledge first

## Build posture

This pack assumes:

- file-only first
- Data Analysis enabled
- stable ION doctrine in Knowledge
- mutable continuity in uploaded working bundles
- encrypted vault/archive kept local and outside ChatGPT

- `06_END_TO_END_VALIDATION_RUNBOOK.md` — minimal end-to-end validation loop
- `07_EXAMPLE_BUNDLE_USAGE.md` — how to use the included example bundles
