# KNOWLEDGE PACK README

This directory contains the first curated ION custom GPT Knowledge pack.

It is intentionally compact and stable. Upload these files into GPT Knowledge rather than the full repo.

## Why this pack exists

The GPT needs:
- stable shell law
- bundle doctrine
- resume/export law
- the minimal temporal canon needed to interpret future-bearing continuity correctly

It does not need the full mutable project organism inside Knowledge.
