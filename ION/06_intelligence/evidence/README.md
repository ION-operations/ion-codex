# Evidence Lane

This lane holds bounded evidence extracts when a research carrier needs to separate
raw branch evidence from wider synthesis.

Current branch posture:
- `Thoth` may write directly to `ION/06_intelligence/research/` for complete evidence-backed
  notes.
- `ION/06_intelligence/evidence/` exists as an optional holding lane for narrower extracts,
  tables, or intermediate evidence bundles when a task packet explicitly asks for them.

Do not treat this lane as a second research center.
It is a support lane under the active Codex-led runtime.
