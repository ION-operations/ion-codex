# ION templates

This directory is a live operational template surface. It must exist at the live root, not only inside archived donor packages.

V79 restores the minimum context-authority template floor needed for ION-native context packaging:

- `context/AGENT_CONTEXT_PACKAGE.md`
- `context/ION_CONTEXT_DELTA_RECEIPT.md`
- `context/ION_CONTEXT_LOAD_PROOF.md`
- `agents/ION_CONTEXT_SPECIALIST_RETURN.md`

Templates are not prose decoration. They define lawful packet shapes, proof shapes, and receipt shapes. For ION context work, `MINI/CAPSULE are not primary context authority`; High-Detail Agent Context Packages are.
