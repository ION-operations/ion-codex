# Custom GPT Example Bundles

This directory contains realistic example bundles for the first ION custom-GPT validation loop.

## Included examples

- `working_bundle_example/` — GPT-readable working continuity bundle
- `vault_bundle_example/` — local encrypted-archive shape example
- `generated/working_bundle_example.zip` — ready-to-upload working bundle artifact
- `generated/vault_bundle_example.zip` — illustrative archive artifact

## Recommended first use

Upload `generated/working_bundle_example.zip` to the first ION custom GPT build and run the end-to-end validation runbook.
