# Custom GPT Templates

This directory contains starter templates for the first ION custom-GPT product loop.

Templates here are intentionally minimal and designed to support:

- a working continuity bundle
- a vault continuity bundle
- deterministic export naming
- compact manual/file-only operation first

## Examples

The `examples/` subtree contains a realistic working continuity bundle example, a vault bundle example, and generated zip artifacts for quick builder validation.
