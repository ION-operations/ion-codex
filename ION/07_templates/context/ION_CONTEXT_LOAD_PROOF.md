# ION_CONTEXT_LOAD_PROOF

A valid context-load proof begins with exactly:

```md
### CONTEXT PROOF
```

Required fields:

- package_id:
- role_id:
- carrier:
- files_loaded:
- source_checksums_verified:
- actual_loaded_material_summary:
- stale_surfaces_detected:
- authority_ceiling_acknowledged:
- return_scope:

Invalid proof patterns:

- “I read the context file.”
- “I understand the context.”
- “I will follow the boot.”
- Any answer that does not demonstrate actual content loaded from the package.
