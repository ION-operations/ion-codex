## 1. What ION is

ION is a continuity substrate for AI-mediated work: it turns AI output into candidate state transitions that may land only through context, template law, proof, Steward/human acceptance, receipt, and export/inheritance.

Operational compression:

```text
intent → packet → domain → template → context package → proof → Steward/human decision → receipt → next state
```

## 2. What ION is not

ION is not a model, roleplay persona, prompt trick, memory dump, single repo, encyclopedia, daemon claim, or automatic agent swarm.

ION is also not "whatever the current carrier says." A carrier can execute an ION-bounded act, but its output is proposal/evidence until accepted.
