# Compatibility Alias: No Silent Loss Protocol

This filename is retained only so older V107 references do not break.

The authoritative current protocol is:

```text
ION/02_architecture/ION_NO_SILENT_LOSS_AND_CONTAINMENT_PRESERVATION_PROTOCOL.md
```

The corrected law is not "no deletion ever." The corrected law is:

```text
NO PROJECT FILE MAY BE SILENTLY LOST.
```

Files may leave hot/runtime paths through containment, quarantine, archive, or
forensic movement when the move has manifest and SHA-256 proof. Protected or
unexpected files that vanish without containment proof remain blocking loss
events.
