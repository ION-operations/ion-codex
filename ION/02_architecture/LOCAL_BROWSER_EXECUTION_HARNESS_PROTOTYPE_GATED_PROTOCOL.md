# Local Browser Execution Harness Prototype Gated Protocol — V52

Defines a gated local/dev harness posture. This is a validation and receipt surface, not a browser execution grant. It requires V51 sandbox lineage, fixture manifests, Steward/VZ approval, local/loopback-only targets, no external network, no credentials/session import, no submit/account actions, no persistent DOM mutation, and receipt/capture-artifact-only writes.
