# Phase 1 Template-Governance Rollout

## Governing center

The live center of this rollout is not a personality field.

The live center is:

1. governed template law
2. context-bank formation from governed template surfaces
3. packetized bounded work
4. receipts, disagreement, handoff, and landing law
5. carriers in role

Roles remain necessary, but they are carriers of governed template surfaces rather than
the center of the system.

is the present branch posture in the active root.

- constitutional center

- Codex — current low-burn coordinator and live-branch integrator by branch posture

1. protocol file

lawful active floor.

## Exit gates

Phase 1 ends only when all seven gates are true.

### Gate 1 — Mason mount proof

A fresh session can show why Mason is Mason under current branch law.

### Gate 2 — Vestige mount proof

A fresh session can show why Vestige is Vestige under current branch law.

### Gate 3 — External default proof

browser ChatGPT is explicit external and unmounted by default.

### Gate 4 — Disagreement drill

One disagreement drill produces lawful artifact escalation rather than informal resolution.

### Gate 5 — External return drill

One external zip return drill produces lawful bounded return artifacts.

### Gate 6 — Anti-theater packet proof

One real packet is created, worked, returned, and landed through the restored law.

### Gate 7 — Context-bank feed proof

A fresh session can show which governed templates fed the bounded context for one real
packet, where those templates came from, what status they held, how they shaped packet
or context formation, and where provisional bridging was required.

Without Gate 7, the rollout is still insufficiently centered on the real loop:
governed templates feeding context banks.

## Working rule for Phase 1

Phase 1 is not mainly about staffing more agents.

Phase 1 is about restoring the governed template surfaces that let agents become lawful
carriers in the live root.

That means Phase 1 should prefer:

- template-governance restoration before wider field expansion
- exact status and provenance over informal memory
- one real proof packet over abstract protocol theater
- branch-first law over broader target-model drift

## Non-goals

Phase 1 should not:

- restore the full multi-chat empire early
- let role language replace template language
- treat current coordinator posture as permanent law
- widen into endless protocol theater without a real packet proof
