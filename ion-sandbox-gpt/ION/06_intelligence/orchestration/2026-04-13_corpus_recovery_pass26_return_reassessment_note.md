# Corpus Recovery Pass 26 — Return and Reassessment Layer

## What this pass adds

Pass 26 strengthens the project control kernel by adding the missing layer for widened work:
- when widened work should return to the kernel
- when it should stop and reassess
- how escalation beyond the kernel should be recorded
- worked examples for both paths

## Why it matters

Without this layer, the control kernel can still be weakened by a subtler drift:
lawful start, lawful widening, then indefinite continuation with no explicit return or stop rule.

Pass 26 closes that gap.
