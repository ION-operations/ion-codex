# Atlas base usage contract

## Rule 1
Treat the atlas base as the current most trustworthy starting center, not as final canon.

## Rule 2
Do not invent a replacement for any center until the relevant fractured-core and supporting-line profiles have been checked.

## Rule 3
Do not collapse preserved conflicts into a single winner unless a later evidence pass truly resolves them.

## Rule 4
Use the atlas base to classify every proposed system or claim into one of:
- active current line
- strongest historical center
- continuation witness
- mixed sibling witness
- archaeology/atlas support line
- wrapper witness
- peripheral witness
- unresolved

## Rule 5
When future work begins, start from the atlas base read order before widening into the rest of the corpus recovery tree.
