# Atlas control surface

## Purpose

This is the smallest control surface for future planning and recovery work.

Use this when you need to decide:
- where to start
- which preserved center to consult first
- whether a new idea is actually new
- whether a future packet should be allowed to proceed

## Minimum decisions this surface should produce

For any future work item, this surface should help determine:
- question class
- primary preserved center
- whether conflict already exists
- whether the work posture is consult / compare / recover / bounded extension / bridge repair

## Core warning

The project repeatedly failed by starting from the wrong center and then inventing a local substitute.
This control surface exists to prevent that.

## Output, landing, and orchestration discipline

Future work should leave this surface knowing not only the question class, center, and posture, but also:
- the narrowest lawful answer/output class
- the narrowest lawful landing boundary
- the current process position
- the near / mid / far horizon
- what complete enough means for the current pass
