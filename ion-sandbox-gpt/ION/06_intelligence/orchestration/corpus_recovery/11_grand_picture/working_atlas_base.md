# Working atlas base

## Purpose

This document is the smallest **working atlas base** for future work inside the project.
It is built on top of Milestone 1 of the corpus recovery program.

Use this surface when you need the fastest honest answer to:
- where should I start?
- which preserved center should I consult?
- what tensions must I not flatten?

## What this atlas base is

This is a **provisional stable atlas base**.
It is strong enough to guide future work.
It is not final canon.
It is not permission to erase preserved contradictions.

The packet-gate layer is now complemented by posture guidance, non-widening defaults, and worked gated-start examples so the atlas is easier to use in practice.

## What not to do

Do not:
- treat the current branch as the whole organism
- treat the strongest historical center in one capability as the best center for all capabilities
- flatten preserved conflicts because one line feels cleaner or newer
- invent a replacement before checking whether the relevant center already exists elsewhere
