# Persona Front-Door Correction

This folder records the A4 correction that followed the initial
single-carrier protocol landing.

The correction is narrow:

- Persona is the normal direct user-discourse front.
- Relay is the courier and packetization layer behind Persona.
- Steward is the orchestration layer behind Relay.
- Internal agents may be visible as workflow/audit transcript, but they do not
  all converse with the user by default.

This does not close final persona canon. It sets the right target route for A5
live-use proof.
