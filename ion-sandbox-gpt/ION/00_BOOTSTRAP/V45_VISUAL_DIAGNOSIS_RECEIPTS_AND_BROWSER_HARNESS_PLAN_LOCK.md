# V45 Visual Diagnosis Receipts and Browser Harness Plan Lock

V45 installs the next bounded Visual Agent layer after V44 visual observation packets.

Authority:
- A3 doctrine/runtime-planning authority only.
- Observe / diagnose / report authority only.
- Browser/screenshot/DOM harness is a plan and packetization surface, not unrestricted computer-control authority.

V45 may:
- compose visual observation packets into diagnosis receipts;
- record findings, evidence, before/after references, and recommendations;
- define a bounded local-dev browser/screenshot/DOM harness plan;
- issue implementation-agent hints for human/steward-reviewed patch work.

V45 may not:
- execute credential-sensitive actions;
- submit forms, purchase, delete, or mutate user accounts;
- perform persistent DOM mutation as authority;
- claim production authority;
- bypass Steward/VZ review for high-risk visual actions.
