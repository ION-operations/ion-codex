# V37 Agent Succession Packet Lock

Status: A3 executable succession primitive, not production authority.
Branch: ION-GPT55-SELF-MOUNT.

V37 installs the agent succession packet. A successor packet carries evidence,
obligations, blocked claims, unresolved risks, and continuation instructions
from one mounted agent to a later mounted agent without asserting numerical
identity between them.

A successor agent inherits task continuity and evidence, not personal
persistence.
