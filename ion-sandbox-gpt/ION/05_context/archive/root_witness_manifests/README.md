# Root Witness Manifests

This directory contains witness files that previously lived at the repository
root.

They were moved here during public repository cleanup so the GitHub root can
serve as a clear entry point while preserving historical evidence.

## Moved File Classes

```text
FILES_ADDED_V*.txt
FULL_PROJECT_CONSOLIDATION_RECEIPT_*.txt
```

Moved count: 40 files.

## Why They Are Here

These files are version and consolidation witnesses. They are not the best first
thing a public collaborator should see, but they are still useful evidence.

Historical manifests and receipts may still reference the old root-level paths.
Those references are not rewritten because they record the state of the
repository at the time they were produced.

## Current Location Rule

For current navigation, use this directory:

```text
ION/05_context/archive/root_witness_manifests/
```

Do not delete or rewrite these witnesses without a lifecycle transition receipt.
