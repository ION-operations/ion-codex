# V51 Successor Handoff — Local Browser Execution Sandbox Specification

V51 defines the strict local/dev browser execution sandbox contract required before any future Visual Agent execution harness prototype.

- no live browser execution authority,
- no unrestricted browser control,
- no external network,
- no credentials/session import,
- no form submission, purchases, destructive action, or account operation,
- no persistent DOM mutation,
- no production visual automation.

Successor branch may prototype a local-only execution harness only if it remains behind Steward/VZ gating and follows the V51 required controls.
